IELTS雅思成績計分說明

IELTS雅思分Listening、Reading、Writing、Speaking等四大項，此四項均獨立計分，最後再將所有4個項目的成績求平均，此為最終總分。所以在成績單上同時會印出四個單項之分數及總平均分數，以顯示此考生在各方面的英語能力。

每所學校或機構要求的IELTS雅思成績不太一樣，但大多數的學校會要求至少具備IELTS6.0-6.5程度，而頂尖學校如牛津、劍橋則通常會要求至少7.0，近幾年也有越來越多學校針對寫作項目有特殊入學要求。

IELTS雅思各級分所代表的能力

9
Expert User
熟練的使用者
極佳使用者，能將英語運用自如；不但適切、精確、流利並能完全理解。
8
very good user
非常好的使用者
非常良好使用者，能將英語運用自如，只是偶爾有繼續的錯誤和不適切，在不熟悉的狀況下可能出現誤解。
7
good user
良好的使用者
良好使用者，有能力運用英語，雖在某些情況下有時會發生不精確不適當和誤解，大致可將複雜的英語掌握得不錯。
6
competent user
合格的使用者
合格的使用者，大致能有效地運用英語，雖然有不精確不適當和誤解發生，在熟悉的情況下能使用並理解相當複雜的英語。
5
modest user
適度的使用者
適度使用者，可部份運用英語，在大多數情況下可應付全盤的定義，雖然可能犯下許多錯誤，在本身領域內應可掌握基本的溝通。
4
limited user
有限的使用者
有限使用者，只限在熟悉的狀況下有基本的理解力，在理解與表達上常發生問題，無法使用複雜英語。
3
extremely limited user
非常有限的使用者
極有限使用者，在極熟悉的情況下，只能溝通理解一般程度。
2
internittent user
困難的使用者
除非在熟悉的狀況下，視溝通的需要使用單字或簡短片語傳達最基本的訊息，實際上不可能有真正的溝通，在英文的說寫方面有重大的障礙。
1
non user
完全不能使用者
本質上無法使用英語，可能可說幾個單字，經常會溝通不良。
0
缺席考試者
缺席考試者。
